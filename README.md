# Rocket Space Theme 
  
  **[Rocket Space](https://gurkanakdeniz.github.io/)** is a jekyll theme.

## Features
* Responsive
* Disqus integration
* Syntax highlighting
* Optional post image
* Social icons
* Page for sharing projects
* Optional background image
* Simple navigation menu
* MathJax support
